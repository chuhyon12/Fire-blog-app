# 🔥 소방 블로그 생성기 — PWA 앱

휴대폰 홈화면에 설치해서 앱처럼 쓸 수 있는 블로그 자동화 도구입니다.

---

## 🚀 Vercel 무료 배포 (10분이면 완료!)

### 1단계 — GitHub 가입 & 업로드

1. https://github.com 회원가입 (무료)
2. **New repository** 클릭
3. Repository name: `fire-blog-app`
4. **Create repository** 클릭
5. **uploading an existing file** 클릭
6. 이 폴더 안의 파일들을 모두 드래그 앤 드롭:
   - index.html
   - manifest.json
   - sw.js
   - vercel.json
   - icon-192.png
   - icon-512.png
7. **Commit changes** 클릭

### 2단계 — Vercel 배포

1. https://vercel.com 접속 → **GitHub으로 로그인**
2. **Add New Project** 클릭
3. `fire-blog-app` 선택 → **Import**
4. 설정 건드리지 말고 **Deploy** 클릭
5. 1~2분 후 배포 완료!

→ `https://fire-blog-app-xxx.vercel.app` 주소 생성됨

---

## 📱 휴대폰 홈화면에 설치

### 안드로이드 (크롬)
1. 위 주소를 크롬으로 접속
2. 우측 상단 점 세 개(⋮) → **홈 화면에 추가**
3. **추가** 탭 → 완료!

### 아이폰 (사파리)
1. 위 주소를 사파리로 접속
2. 하단 공유 버튼(□↑) → **홈 화면에 추가**
3. **추가** 탭 → 완료!

설치 후엔 앱 아이콘 누르면 바로 실행돼요 🔥

---

## 🔑 API 키 설정

1. https://console.anthropic.com 접속
2. 구글 계정으로 로그인
3. 왼쪽 메뉴 **API Keys → Create Key**
4. 생성된 `sk-ant-...` 키 복사
5. 앱 상단 입력창에 붙여넣기
6. **자동 저장됨** — 다음번엔 다시 입력 안 해도 돼요!
